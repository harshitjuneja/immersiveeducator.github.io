A UNITY based immersive (VR) gaming platform for android to help kids aged 8 to 14 years learn programming while having fun solving challenges.
Tracks :
The three tracks planned are:
  • Conditional statements (if-else)
  • Sorting (Selection Sort)
  • Searching (Binary Search)

    • Features :
    Virtual reality for education: Apk for android generated for viewing (initial support for) in the easily available Google Cardboard.
    Learning while gaming: Story based challanges.
